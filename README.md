# Tujuan Repository

Repository ini dibuat sebagai dokumentasi hasil praktikum dan tugas Ujian Tengah Semester (UTS) mata kuliah Pengantar Data Science. Seluruh notebook disusun mengikuti modul praktikum sehingga dapat menunjukkan proses pembelajaran mulai dari pengolahan data, pembersihan data, analisis statistik, visualisasi, preprocessing machine learning, hingga implementasi model prediksi sederhana menggunakan Python.

---
# Daftar Praktikum

## Pertemuan 1 — Pengenalan Data Science

Materi pada pertemuan pertama berisi pengenalan Data Science, penggunaan Google Colab, dasar Markdown, serta pengenalan Python sebagai bahasa pemrograman yang digunakan selama praktikum.

---

## Pertemuan 2 — Struktur Data Python, NumPy, dan Pandas

Praktikum ini membahas struktur data dasar Python seperti list, tuple, dictionary, dan set. Selain itu dipelajari penggunaan library NumPy dan Pandas untuk membaca dataset, melakukan filtering, operasi groupby, dan analisis sederhana menggunakan dataset Titanic.

---

## Pertemuan 3 — Data Cleaning

Materi berfokus pada proses pembersihan data (data cleaning) sebelum dilakukan analisis. Tahapan yang dilakukan meliputi:

- eksplorasi dataset,
- penanganan missing values,
- menghapus data duplikat,
- normalisasi data,
- penanganan outlier menggunakan metode IQR Fence,
- validasi hasil cleaning,
- menyimpan dataset yang telah dibersihkan,
- serta mengambil data dari REST API menggunakan Python.

---

## Pertemuan 4 — Statistika Dasar

Pada praktikum ini digunakan Dataset Iris untuk mempelajari statistik deskriptif dan visualisasi data.

Analisis yang dilakukan meliputi:

- statistik deskriptif,
- histogram dan KDE,
- boxplot,
- violin plot,
- scatter plot,
- korelasi Pearson,
- heatmap korelasi.

---

## Pertemuan 5 — Dashboard Visualisasi Data

Praktikum ini menggunakan Heart Disease UCI Dataset untuk membuat dashboard visualisasi statis.

Dashboard terdiri dari beberapa jenis visualisasi, yaitu:

- Bar Chart
- Histogram
- Boxplot
- Scatter Plot

Setiap visualisasi disertai interpretasi menggunakan pendekatan What – So What – Now What

---

## Pertemuan 6 — Data Preparation

Pada pertemuan ini digunakan Dataset Titanic untuk mempelajari proses persiapan data sebelum digunakan dalam machine learning.

Tahapan preprocessing yang dilakukan meliputi:

- Exploratory Data Analysis (EDA)
- Handling Missing Values
- One-Hot Encoding
- Train-Test Split
- Feature Scaling menggunakan StandardScaler

---

## Pertemuan 7 — Linear Regression

Pertemuan terakhir membahas implementasi machine learning sederhana menggunakan Linear Regression.

Dataset yang digunakan merupakan dataset sintetis prediksi gaji dengan beberapa variabel seperti pengalaman kerja, tingkat pendidikan, dan kota. Praktikum meliputi proses preprocessing, pelatihan model, evaluasi performa model, serta visualisasi hasil prediksi.
